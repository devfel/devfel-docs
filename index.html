DevFel's Documentation & Quick Guide

This page was created to be a quick guide for most of the usual commands and functions used by developers when dealing with several languages and frameworks.

Initial Basic Configuration

To load a CSS file, add the following to the header tag in you html

<!-- Own CSS Style -->
<link rel="stylesheet" href="./css/main.css" />

To load a Javascript file, add the following to the end of the body tag

<!-- Own JS Script -->
<script src="./js/main.js"></script>
